## Quick Navigation

**시작하기:**
- 새로운 프로젝트라면 → [Executive Summary](01-executive-summary.md)부터 읽기

- 기술적 세부사항이 필요하다면 → [System Architecture](04-architecture/system-architecture.md)

- **Product Manager**: Executive Summary → Product Overview → Features → Roadmap
