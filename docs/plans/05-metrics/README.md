## Key Concepts

-   **North Star Metric:** Active Learning Groups (ALG) - groups with >3 weekly interactions
-   **Activation:** Assessment completion, matching success, first project start
-   **Retention:** Day 30 retention, WAU, group survival rate
-   **Outcomes:** Project completion, portfolio generation, NPS
