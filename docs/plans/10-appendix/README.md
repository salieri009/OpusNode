## Quick Reference

### Glossary (Key Terms)
-   **Assessment:** AI interview for skill evaluation
-   **Matching:** Algorithmic team formation based on complementary skills
-   **Curriculum:** Personalized learning roadmap
-   **Workspace:** Group project collaboration environment
-   **Portfolio:** Auto-generated career showcase from completed projects
-   **Vector DB:** Database for storing AI embeddings (Milvus/Pinecone)
-   **LLM:** Large Language Model (GPT-4, Claude 3.5)

### Development Workflow Highlights
-   **Sprint:** 2-week cycles
-   **PR Reviews:** 1 approval for `dev`, 2 for `main`
-   **Code Coverage:** 80% minimum, 95% for critical paths
-   **Incident Response:** P0 (immediate), P1 (1hr), P2 (4hrs), P3 (next business day)
